# Recommend me a movie for tonight

An older puppeteer script, using IMDb and DuckDuckGo to recommend you a given number of movies for tonight.

## How to run it?

Install npm packages:

```
yarn
```

Run script:

```
node index.js
```

Example output:

```
I recommend you the following 3 movies from IMDB Top 250:
----------------------------------------------------------
#93. Capernaum (2018)
#46. Modern Times (1936)
#90. Eternal Sunshine of the Spotless Mind (2004)

Watch them on Netflix:
------------------------
• Capernaum on Netflix: https://www.netflix.com/title/81002571
• Modern Times on Netflix: https://www.netflix.com/title/60028129
• Eternal Sunshine of the Spotless Mind on Netflix: https://www.netflix.com/title/60034545
```
